This page has a few moving parts:

  1. The containing div is relatively positioned in relation to the whole web page, so it scrolls up and down as expected.
  2. The div contains an image, plus two smaller divs nested within it, which are the hotspots.
  3. The hotspots are absolutely positioned in relation to the containing div, so they are fixed within it.
  4. Hotspots A and B each have their own unique coordinates.
  5. Hotspots A and B are each divs with rounded corners, so they appear as circles instead of squares
  6. Width dimensions are set by percentages, not pixels, for device scaling.
